# jquery-pc

> 一个jquery-webpack的项目

## 项目初始化步骤

``` bash

# 安装nodejs环境

# 全局安装webpack
    命令: (sudo) npm install -g webpack

# 全局安装webpack-dev-server
    命令: (sudo) npm install -g webpack-dev-server

# 在项目根目录执行npm初始化
    命令: npm install

# 启动项目
    开发模式: npm run dev
    生产模式: npm run build

```
